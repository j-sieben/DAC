# install_scripts

Shared SQL*Plus installation helpers.

## Layout

- `install`: launcher and generic component install scripts.
- `scripts/check`: existence checks used by helper scripts.
- `scripts/helper`: reusable object creation, cleanup, and compile helpers.
- `scripts/make`: DDL fragments called by helpers.
- `scripts/packages`: helper package sources.
- `scripts/spool`: output formatting scripts.
- `scripts/tables`: helper table sources.
