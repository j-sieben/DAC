# AI Project Context: Berechtigungsmatrix

This folder stores working context for AI-assisted development of the standalone
`berechtigungsmatrix` project. It is intentionally local to this project so the
DAC implementation can be understood without relying on the surrounding KISMON
repository context.

Use these files first when resuming work:

- `project-summary.md` - current architecture, core concepts, and object map.
- `decisions.md` - important design decisions and rationale.
- `workflows.md` - install, cleanup, refresh, and test workflows.
- `testing.md` - current test suite intent and known expectations.
- `open-notes.md` - caveats and follow-up items.
